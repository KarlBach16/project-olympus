# Olympus 작업일지

## 2026-06-15

### 목표

`MVP.md` 계획을 바탕으로 Godot에서 첫 플레이 가능한 프로토타입 루프를 만든다.

### 구현한 것

- 기본 Godot 프로젝트 구조를 만들었다.
  - `scenes/`
  - `scripts/`
  - `sprites/`
  - `ui/`
- 아킬레우스를 플레이어 캐릭터로 추가했다.
- WASD와 방향키 이동을 추가했다.
- 플레이어를 따라가는 카메라를 추가했다.
- 간단한 필드 배경을 추가했다.
- 첫 번째 적으로 키클롭스를 추가했다.
- 두 번째 잡몹으로 하피를 추가했다.
- 적이 플레이어를 향해 이동하도록 만들었다.
- 접촉 피해를 추가했다.
- 플레이어 HP UI를 추가했다.
- 사망 상태와 `R` 키 재시작을 추가했다.
- 근접 자동 공격을 추가했다.
- 적 HP, 피격 깜빡임, 사망 처리를 추가했다.
- 플레이어 주변에 적이 계속 생성되도록 스포너를 추가했다.
- XP 오브를 추가했다.
- XP 수집, 레벨, XP UI를 추가했다.
- 레벨업 시 랜덤 강화가 자동 적용되도록 만들었다.
  - `+1 Damage`
  - `+10% Attack Speed`
  - `+10% Move Speed`
- 강화 알림 문구를 추가했다.
- 하피는 스프라이트만 위아래로 움직여 떠 있는 느낌을 주도록 만들었다.
- 생존 시간 UI를 추가했다.
- 테스트용으로 30초 후 키클롭스 보스 이벤트가 시작되도록 만들었다.
- 보스 등장 경고 문구를 추가했다.
- 전용 이미지와 높은 능력치를 가진 키클롭스 보스 씬을 추가했다.
- 보스 처치 시 승리 조건을 추가했다.
- 승리 후 `R` 키로 재시작할 수 있게 만들었다.
- 아킬레우스 정면 2프레임 걷기 애니메이션을 추가했다.

### 현재 게임 루프

1. 필드에 아킬레우스가 등장한다.
2. 플레이어가 키보드로 아킬레우스를 움직인다.
3. 키클롭스와 하피가 플레이어 주변에서 생성된다.
4. 적들이 아킬레우스를 추적한다.
5. 아킬레우스가 가까운 적을 근접 베기로 자동 공격한다.
6. 적이 죽으면 XP 오브를 떨어뜨린다.
7. 아킬레우스가 XP 오브를 수집하고 레벨업한다.
8. 레벨업 시 랜덤 강화 하나가 즉시 적용된다.
9. 플레이어는 적과 접촉하면 피해를 받는다.
10. 플레이어는 죽을 수 있고, 죽은 뒤 재시작할 수 있다.
11. 30초가 지나면 일반 스폰이 멈추고 키클롭스 보스가 등장한다.
12. 보스를 처치하면 승리 문구가 표시되고 재시작할 수 있다.

### 튜닝 메모

- 기존 근접 공격 범위가 너무 짧아 적을 치다가 접촉 피해를 받는 경우가 많았다.
- 플레이어 공격 범위를 `95`에서 `135`로 늘렸다.
- 적 이동 속도를 낮췄다.
  - 키클롭스: `90 -> 70`
  - 하피: `135 -> 110`
- 보스 등장 시간은 현재 테스트용으로 `30초`다.
- 최종 MVP 기준에서는 보스 등장 시간을 `300초`로 바꿀 수 있다.

### 에셋 메모

- 아킬레우스는 정면 걷기 2프레임을 사용한다.
  - `sprites/achilles_walk_1.png`
  - `sprites/achilles_walk_2.png`
- `achilles_walk_1.png`는 현재 alpha 채널이 없다.
- 실행 중 체크무늬 배경이나 불필요한 배경이 보이면 `achilles_walk_1.png`를 투명 PNG로 교체해야 한다.
- 키클롭스, 하피, XP 오브, 키클롭스 보스 이미지는 `sprites/`에 import 되어 있다.
